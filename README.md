# IndiaSalaryCoach

AI-powered salary analysis, offer comparison, resume optimization and career guidance platform.

## Tech Stack

- Java 17
- Spring Boot 3
- PostgreSQL
- JWT Authentication
- Maven

## Features

- Salary Analysis
- Offer Comparison
- Career Reports
- Resume Builder
- Resume Scoring
- AI Career Guidance
- Subscription Plans

## Run Locally

Set environment variables:

DATABASE_URL
JWT_SECRET

Then:

mvn spring-boot:run
